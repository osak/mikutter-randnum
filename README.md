# mikutter-randnum

## What's this
適当な乱数を埋め込みます。

Install
-------
`osa_k_store <http://github.com/osak/osa_k_store>`_ を導入してみっくストア経由で入れるか，もしくは ::

    git clone git://git.github.com/osak/mikutter-randnum.git ~/.mikutter/plugin/randnum

How To
------
%rand%とツイート本文に書くと、投稿時に適当な乱数に置換されます。
